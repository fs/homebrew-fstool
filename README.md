homebrew-fstool
===============

Installation
---

```shell
brew tap fs/fstool
brew install fs-tool
```

You can also install via URL:

```
brew install https://raw.github.com/fs/homebrew-fstool/master/fs-tool.rb
```

Docs
----

`brew help`, `man brew`, or the Homebrew [wiki](http://wiki.github.com/mxcl/homebrew).
